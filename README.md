# omnixys-media

Omnixys shared media/storage package with S3/MinIO support via aiobotocore.

## Installation

```bash
pip install omnixys-media
```

## Features

- S3/MinIO compatible storage service
- Async file upload and retrieval
- Storage result models
- Dependency injection support

## Usage

```python
from omnixys_media import S3StorageService, MediaProvider, StorageResult
```
